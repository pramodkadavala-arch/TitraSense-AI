# TitraSense AI

## AI-Powered Smart Auto Titrator for Real-Time Reaction Endpoint Prediction

---

# Project Overview

TitraSense AI is an intelligent laboratory automation platform designed to improve the accuracy, affordability, and accessibility of analytical chemistry experiments.

The system automates titration by integrating spectral sensing, AI-assisted endpoint prediction, precision liquid dosing, and real-time monitoring into a single portable platform.

Unlike conventional laboratory titration methods that rely on manual observation and expensive analytical instruments, TitraSense AI provides a low-cost smart alternative suitable for educational institutions, research laboratories, pharmaceutical industries, and chemical laboratories.

---

# Problem Statement

Manual titration depends on human observation to identify reaction endpoints.

This approach often leads to:

- Human errors
- Endpoint overshooting
- Poor reproducibility
- Time-consuming analysis
- Expensive commercial alternatives

There is a growing need for an intelligent, affordable, and automated titration platform capable of predicting reaction endpoints in real time while minimizing human intervention.

---

# Proposed Solution

Our solution combines embedded systems, spectroscopy, artificial intelligence, and IoT technologies into one integrated platform.

The system continuously monitors the reaction using an AS7341 spectral sensor.

An ESP32-S3 controller processes the sensor data and controls:

- Peristaltic Pump
- Magnetic Stirrer
- LCD Display
- Voice Assistant
- Mobile Application

As the reaction approaches completion, the AI algorithm predicts the endpoint and automatically slows titrant addition to improve accuracy.

The entire experiment can be monitored through a mobile application.
